### Sections to populate

1. **Theme axes — popularity survey (2024–2026).**
   - Fantasy (high, dark, mythic, folklore).

2. **Signal sources.** Steam tag co-occurrence and revenue, top-grossing strategy titles by theme, breakout 2024–2026 indies, streamer topic preferences, merch/cosplay indicators.
3. **Era-spine candidates for this game.**
   - **Candidate A — classical→posthuman (current).** Primal → Mudbrick → Iron → Spires → Alchemical → Steamheart → Atomic → Signal → Synthetic → Stellar → Ascendant.
   - **Candidate B — myth-forward.** Each age is a world-mythology (Sumerian, Norse, Mesoamerican, Shinto, Afro-futurist, posthuman-mythic). Gives every age a marketing-memorable visual identity.
   - **Candidate C — collapse-and-rebuild.** Ages alternate bloom and collapse. Zombies/plague/cataclysm as recurring antagonist motifs. Hits the "what humanity abandoned" direction from [CONCEPT §4.7](../CONCEPT.md).
   - **Candidate D — parallel worlds.** Each age is a different civilization's alt-timeline (what if Rome never fell, what if Song-dynasty industrialized, etc.). Strong anti-commodity play.
   - **Candidate E — themed-overlay hybrid.** Keep the civilizational arc, allow seasonal themed overlays ("Season of the Pirates" reskins ages 3–5). Lets us chase trend themes without core instability.
4. **Trope-fatigue analysis.** Which themes are over-represented in 2024–2026 strategy games (and therefore risky as core spine).
5. **Zombies / pirates / mythic — specific deep dives per user interest.**
6. **Silhouette-forward art fit.** Each candidate era spine scored for compatibility with the [CONCEPT §5.3](../CONCEPT.md) leading art direction.
